---+!! Empty Plugin

%SHORTDESCRIPTION%

Sometimes it's useful for an admin to be able to perform an immediate check on the configuration from within a Foswiki page. This plugin can be used to help debug a configuration by providing feedback on configuration errors, warnings and messages in a wiki page.

*WARNING* this plugin _can be run by any admin_ and _may reveal sensitive configuration information_. It should *not* be left active in a production wiki (though it can be left installed, so long as it is safely disabled).

---++ Usage
Add the [[VarCONFIGURE_CHECK][%<nop>CONFIGURE_CHECK%]] macro to any topic to get a report of the messages generated by =configure= checkers.
